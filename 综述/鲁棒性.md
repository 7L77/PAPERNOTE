# 鲁棒性

## NTK 
### ZCPRob（2025 ICLR）
ZERO-COST PROXY FOR ADVERSARIAL ROBUSTNESS EVALUATION
提出了一种鲁棒神经正切核（Robust NTK）理论。它的核心贡献是利用 NTK 理论在初始化阶段直接评估网络对对抗扰动的抵抗力，从而避免了在搜索过程中生成昂贵的对抗样本。

### 1（2024 ICLR） 
Robust NAS under adversarial training: benchmark, theory, and beyond
一种基于 NTK 的多目标对抗训练泛化理论。作者提出了一种名为 “双重鲁棒 NTK (Twice-robust NTK)” 的方法，用极其严谨的数学推导证明了通过特定的 NTK 变体，可以同时预测架构的干净准确率（Clean Accuracy）和鲁棒准确率（Robust Accuracy）。

### NTK+线性区域 （2024 AAAI）
Neural Architecture Search via Trainless Pruning Algorithm (TTNAS)
使用贝叶斯方法（True-Skill算法）来评估架构。在它的底层指标中，它直接提取了 NTK 的频谱特征（Spectrum of NTKs） 和 输入空间内线性区域的量化数量 作为评估架构质量的核心代理指标。
